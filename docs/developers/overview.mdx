### Building Pieces

Building pieces allows you to customize the project to perfectly suit your needs. You can follow our tutorials to learn how to create pieces and use them for your own purposes or share them with the community.

### How to Contribute

This section aims to provide guidance on how you can contribute to the Activepieces project. Whether you're a developer, designer, or user with suggestions, we value and appreciate your input. Here are the steps to get involved:

1. **Report bugs or request features:** If you encounter any issues or have ideas for new features, please open an issue on our GitHub issue tracker. You can access it [here](https://github.com/activepieces/activepieces/issues).

2. **Suggest improvements or report documentation issues:** We welcome your suggestions for enhancing our project or if you come across any problems with the documentation. Open an issue on the GitHub issue tracker to let us know.

3. **Build new pieces:** Building new pieces from the [community](https://community.activepieces.com/c/feature-requests/9/l/latest?order=votes) sorted by votes is vital for the growth of our project. We encourage you to contribute by creating new pieces that align with the project's objectives.

4. **Create tutorials:** We are always looking for new tutorials to add to our documentation. If you have any ideas for tutorials, please let us know by opening an issue on the GitHub issue tracker.

5. **Roadmap:** Check our [Roadmap](https://github.com/orgs/activepieces/projects/53) to see what we are working on and what we have planned for the future. If you have any suggestions for new features or improvements, please let us know by opening an issue on the GitHub issue tracker.
