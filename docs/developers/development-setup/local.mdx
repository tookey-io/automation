## Prerequisites
- Node.js v18+
- npm v9+

## Instructions

1. Setup the environment
```bash
node tools/setup-dev.js
```

2. Start the environment

This command uses will start activepieces with sqlite3 and in memory queue.

```bash
npm start
```

3. Go to ***localhost:4200*** on your web browser and sign in with these details:

Email: `dev@ap.com`
Password: `12345678`
