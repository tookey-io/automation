## Actions


      <Card title="Create or Update Contact " >
        Create or update an existing contact

        <Expandable title="Properties">
          
<ResponseField name="Project API key" required type="Secret_text">
Your project API key
</ResponseField>
            
<ResponseField name="Email" required type="Short_text">
Email address of the user. Mandatory if "SMS" field is not passed in "attributes" parameter. Mobile Number in SMS field should be passed with proper country code. For example: {"SMS":"+91xxxxxxxxxx"} or {"SMS":"0091xxxxxxxxxx"}
</ResponseField>
            
<ResponseField name="External ID"  type="Short_text">
Pass your own Id to create a contact.
</ResponseField>
            
<ResponseField name="Attributes"  type="Object">
Pass the set of attributes and their values. The attribute's parameter should be passed in capital letter while creating a contact. These attributes must be present in your SendinBlue account. For eg:
      {"FNAME":"Elly", "LNAME":"Roger"}
</ResponseField>
            
<ResponseField name="Email Blacklisted?"  type="Checkbox">
Set this field to blacklist the contact for emails (emailBlacklisted = true)
</ResponseField>
            
<ResponseField name="SMS Blacklisted?"  type="Checkbox">
Set this field to blacklist the contact for SMS (smsBlacklisted = true)
</ResponseField>
            
<ResponseField name="List IDs"  type="Array">
Ids of the lists to add the contact to.
</ResponseField>
            
<ResponseField name="SMTP Blacklist Sender"  type="Checkbox">
transactional email forbidden sender for contact. Use only for email Contact ( only available if updateEnabled = true )
</ResponseField>
              
        </Expandable>
      </Card>
