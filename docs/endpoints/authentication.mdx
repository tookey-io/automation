<Tip>

API keys are generated under the platform dashboard at this moment, which is only available in the Platform and Enterprise editions, contact sales@activepieces.com for more information.
</Tip>


The API uses `API keys` to authenticate requests. You can view and manage your API keys from the Platform Dashboard. 

**Specifiy Project**:

The Platform API key can be used on behalf of a project by specifying `X-PROJECT-ID` in the header with the Activepieces project Id. This header is required for resources that are located under a project (such as Flow, Flow Run, and Connections).
