Activepieces uses RBAC in its projects. In each project, there can be many flows and users. These users get specific roles that say what they can do in the project.

The supported roles in Activepieces are:

- **Admin:**
  - View / Edit Flows
  - View Runs
  - Manage Users

- **Editor:**
  - View / Edit Flows
  - View Runs

- **Viewer:**
  - View Flows
  - View Runs
